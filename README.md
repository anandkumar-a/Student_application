# Student Management Application

## 📌 Project Overview

The Student Management Application is a web-based application developed to manage student information efficiently. It allows users to perform various operations such as adding, updating, viewing, and deleting student records.

The project was developed to understand the fundamentals of backend development, database connectivity, and CRUD operations using Java technologies.

## 🚀 Features

* Add new student records
* View all students
* Update existing student details
* Delete student records
* Search students by ID or name
* Database integration for persistent storage

## 🛠 Technologies Used

* Java
* Spring Framework / Spring Boot
* MySQL
* Hibernate / JPA
* Maven
* REST API
* Postman
* IntelliJ IDEA / Eclipse


## 📊 Student Details Managed

The application stores information such as:

* Student ID
* Student Name
* Age
* Gender
* Department
* Email Address
* Phone Number
* Address

## ⚙️ API Endpoints

| Method | Endpoint         | Description            |
| ------ | ---------------- | ---------------------- |
| GET    | `/students`      | Retrieve all students  |
| GET    | `/students/{id}` | Retrieve student by ID |
| POST   | `/students`      | Add a new student      |
| PUT    | `/students/{id}` | Update student details |
| DELETE | `/students/{id}` | Delete a student       |

## ▶️ How to Run the Project

1. Clone the repository.
2. Configure the database credentials in `application.properties`.
3. Run the MySQL server.
4. Build the project using Maven.
5. Start the application.
6. Test the APIs using Postman or access the application through the browser.
